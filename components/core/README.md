# @monorepo/core

Core shared components for the monorepo.

## Components

- `Button`: A simple button component
- `Card`: A card container component

## Usage

```javascript
import { Button, Card } from "@monorepo/core";

// Use in your app
<Card title="My Card">
  <Button title="Click Me" onPress={() => console.log("pressed")} />
</Card>;
```
